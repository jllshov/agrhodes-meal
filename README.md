# A.G. Rhodes Atlanta — Meal Check-In (Vercel)

Fast mobile-optimized meal check-in page for A.G. Rhodes Atlanta employees.

## Setup

1. Push this folder to GitHub
2. Connect to Vercel
3. Deploy

## URLs

- Check-in: https://agrhodes-meal.vercel.app
- Feedback: (Google Apps Script URL)?page=feedback  
- Admin: (Google Apps Script URL)?page=admin

## Backend

All data is stored in Google Sheets via Google Apps Script.
The Apps Script URL is hardcoded in index.html.
